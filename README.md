# Spooner Labs

Personal site for publishing experiments and thoughts in systems, technology, organizations, and whatever else is worth exploring.

**Live at:** [spoonerlabs.com](https://spoonerlabs.com)

## Stack

Plain HTML, CSS, and minimal JavaScript. No framework. Deploys automatically to Cloudflare Pages from `main`.

## Structure

```
/
├── index.html
├── about.html
├── contact.html
├── speaking.html
├── site.css
├── logo.svg
├── experiments/
│   └── index.html
└── projects/
    └── index.html
```

## Publishing

Push to `main` → Cloudflare Pages deploys automatically.
